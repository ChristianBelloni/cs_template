# C# Rust bindings template

## Prerequisites

- [cargo generate](https://github.com/cargo-generate/cargo-generate)

## Usage
```console
cargo generate --git https://github.com/ChristianBelloni/cs_template.git
```
```console
make install-dep
```
Build all targets
```console
make
```
Generate bindings
```console
make csharp
```

## Features
 - Android targets
 - IOs Macos targets
 - Windows target
